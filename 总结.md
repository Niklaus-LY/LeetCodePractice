#### 链表

- 哑节点方法可以处理边界问题，类比迷宫问题增加多一圈围墙。
- 有序问题往往可以考虑达到nlogn的复杂度
- 链表的引用要避免成环



#### 通用

- 迭代考虑递归，递归考虑尾递归。

- 乘法和除法可以用移位代替，速度更快 `a / 2` —> `a >> 1 `

- 加法防止溢出
  
  - 二分查找中`(l + r) / 2 (其中l<r)`写成 `l + (r - l) / 2`-->进一步优化`l + r >> 1`
  
- 交换两个数

  ```C++
  a = a ^ b;
  b = a ^ b;
  a = a ^ b;
  ```
